# Embedded Systems & ECE Project Portfolio

A collection of hands-on engineering projects spanning embedded systems, real-time OS deployment, automotive protocols, and vehicle HMI development — built as part of coursework, club work, and independent exploration.

---

## Projects

### Qt/QML Vehicle Dashboard — `snerith_5`

A full vehicle dashboard UI built with Qt 6.11.1 and CMake, targeting embedded deployment on a QNX/Raspberry Pi 5 platform. The interface is designed with ratio-based scaling, animated widget transitions, and a modular component architecture covering speed, steering, battery, lighting controls, and a settings panel.

**Stack:** Qt 6.11.1 · QML · CMake · QNX (target)

---

### CAN Bus Communication & Analysis

End-to-end CAN bus implementation on the Infineon AURIX TC334 Lite Kit, from firmware-level frame transmission using the onboard transceiver to PC-side reception and analysis via a USB-CAN FD adapter. Includes Python-based tooling for bus monitoring and frame inspection.

**Stack:** AURIX TC334 · iLLD · CAN FD · Python · python-can

---

### STM32 Embedded Development

Peripheral bring-up and firmware development on the STM32F446RE Nucleo platform, covering UART communication and GPIO-based input handling.

**Stack:** STM32F446RE · STM32CubeIDE · HAL · C

---

### QNX RTOS on Raspberry Pi 5

Setup and deployment pipeline for QNX SDP 8.0 on Raspberry Pi 5 hardware, including BSP extraction, Momentics IDE configuration, and filesystem image preparation for embedded target deployment.

**Stack:** QNX SDP 8.0 · QNX Momentics IDE · Linux · Raspberry Pi 5

---

## Areas of Focus
